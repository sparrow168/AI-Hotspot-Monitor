## 📱 日常使用流程

### 每日必做（5 分钟）

#### 步骤 1：查看热点推送

1. **打开网页**：https://sparrow168.github.io/AI-Hotspot-Monitor/
2. **查看当日热点**：页面会显示匹配关键词的热点新闻
3. **筛选感兴趣的内容**：
   - 按平台筛选（抖音/知乎/B站等）
   - 按关键词分类筛选（AI一人公司/超级个体等）

#### 步骤 2：标记选题灵感

发现感兴趣的内容时：

| 标记 | 操作 |
|------|------|
| 🔥 高热度 | 点赞收藏，作为重点选题 |
| 📝 待深入 | 记录到选题库，后续深度挖掘 |
| 💡 灵感 | 记录观点，用于内容创作 |

#### 步骤 3：制定创作计划

根据热点，规划今日内容：

```
今日选题计划：
├── [ ] 制作关于 xxx 的视频
├── [ ] 撰写关于 xxx 的图文
└── [ ] 直播素材收集
```

## 🔑 关键词管理

### 查看当前关键词

访问配置文件：
👉 https://github.com/sparrow168/AI-Hotspot-Monitor/blob/main/TrendRadar/config/frequency_words.txt

### 添加新关键词

#### 方法 1：通过 GitHub 网页修改

1. 打开配置文件
2. 点击 **编辑按钮**（铅笔图标）
3. 在对应分类下添加新关键词
4. 点击 **Commit changes**

#### 方法 2：告诉我，我来添加

直接在这里告诉我你想添加的关键词，我会帮你更新。

### 关键词语法

```
单个词：
  华为

多个词（或者关系）：
  华为
  鸿蒙
  任正非

正则匹配：
  /华为|鸿蒙|任正非/

带别名：
  /深度求索|幻方量化/ => DeepSeek

限制数量：
  @5  # 最多显示5条

排除词：
  !水果  # 排除包含"水果"的内容
```

### 常用关键词建议

## ❓ 常见问题

### Q1: 收不到热点推送？

**A:** 目前仅支持网页查看，微信推送待配置。

### Q2: 关键词不匹配？

**A:** 检查关键词语法：
- 确保没有拼写错误
- 使用正则表达式 `/词1|词2/` 匹配多个词
- 尝试更通用的关键词

### Q3: 热点太少？

**A:** 可能原因：
1. 关键词太具体 → 改为更通用的词
2. 平台未启用 → 检查 `config.yaml`
3. 热点较少 → 等待或添加更多关键词

### Q4: 如何删除关键词？

**A:** 在 GitHub 文件中删除对应行，提交即可。

### Q5: 页面显示延迟？

**A:** GitHub Actions 每小时运行一次，可能有 5-10 分钟延迟。
